# DSX‑Connect

A modular, integration‑agnostic framework for orchestrating file scanning and remediation at scale.

- Reusable scanning core (queues + workers)
- Pluggable DSX‑Connectors for each repository/service
- Event‑driven, fault‑tolerant, and cloud‑portable
